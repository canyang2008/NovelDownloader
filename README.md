# NovelDownloader
## 介绍
一个有多种下载模式的小说下载器，支持多种平台的小说内容提取和多种方式保存，并可以下载封面与插图。    
## 功能
- 搜索下载  
- 群组，用户：将下载的文件归类至对应用户和群组当中  
- 插图下载  
- 下载模式：Chrome，API，Requests  
- 网站支持：
  - 番茄  
  - 起点  
  - 笔趣阁(www.biqugequ.org)  
- 格式支持：
  - txt  
  - json  
  - base  
## 一般操作
下载前检查：
  - 首先是否已有Chrome浏览器  [官网](https://www.google.cn/intl/zh-CN/chrome/)
  - 是否已经登录，本程序**不可下载未解锁的章节**
  - 下载之前需要获取小说相关链接，然后输入到程序中。(链接支持目录页和分享链接)  
    - 目录页：浏览器上页面含有名称和封面图片的  
    - 分享链接：在平台APP阅读页的(系统)分享链接，然后打开，搜索栏就会有长长的字符串，这个就是目标链接  
  - 也可以通过搜索功能下载，但现仅支持部分下载模式的搜索功能  
## 注意事项
- 下载有时因网站的反爬虫机制有时会弹出验证码，需自行解决  
- **此下载器无法下载未解锁的章节！**

## 关于反病毒误报说明
本程序使用 nuitka 进行打包，这与某些病毒的运行方式类似，所以容易误报。  
如果对本程序不放心，请使用其他同类型的产品。
